# Adaptive Core Types

This project contains common types used by multiple Functional Clusters as part of their public interfaces.

## 1. Folder structure

The folder structure is:
```
core-types
├── files
│   ├── cmake
│   └── pkgconfig
├── include
│   └── public
│       └── ara
│           └── core
│               └── internal
└── test
```
