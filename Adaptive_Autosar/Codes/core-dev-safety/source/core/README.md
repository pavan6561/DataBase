# Adaptive Core

This project contains core functionality of the Adaptive Platform.

## 1. Folder structure

The folder structure is:
```
core
├── files
│   ├── cmake
│   └── pkgconfig
├── include
│   └── public
│       └── ara
│           └── core
└── src
    └── test
```
